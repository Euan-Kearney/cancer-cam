# cancer-cam
Skin cancer detection model and application.

This project is licensed under CC BY-NC-SA 4.0. Training data citation: Tschandl, Philipp. 2018. “The HAM10000 Dataset, a Large Collection of Multi-Source Dermatoscopic Images of Common Pigmented Skin Lesions.” Harvard Dataverse. https://doi.org/10.7910/DVN/DBW86T.

It is intended for non-commercial research and educational purposes only.
The software provides probabilistic estimates for skin lesion analysis and is not a medical device or diagnostic tool.
